# Shell Layout Component

**Parent**: `knowledge-ops-workbench-ux-breakdown.md`
**Epic**: 0. Shell Foundation
**Points**: 2
**Dependencies**: None

---

## Purpose

Three-pane responsive shell with Context Bar (left), Main Canvas (center), and Inspection Panel (right).

---

## Scope

Create the root layout component that establishes the three-pane grid structure. This is the foundation all views render into.

---

## Files to Create

| File | Purpose |
|------|---------|
| `src/shell/Shell.tsx` | Root layout component with three-pane grid |
| `src/shell/Shell.module.css` | Responsive grid styles |

---

## Behavior

### Layout Structure

```
┌──────────────────────────────────────────────────────────────────┐
│ Header: logo, org selector, user menu, settings                  │
├──────────┬───────────────────────────────────────┬───────────────┤
│ Context  │ Main Canvas                           │ Inspection    │
│ Bar      │ (swaps by view)                       │ Panel         │
│ (left)   │                                       │ (right)       │
├──────────┴───────────────────────────────────────┴───────────────┤
│ Status Bar: collection, model, tokens, agent, mode               │
└──────────────────────────────────────────────────────────────────┘
```

### Responsive Breakpoints

| Viewport | Context Bar | Inspection Panel |
|----------|-------------|------------------|
| < 768px | Icon strip only (collapsed) | Hidden (icon trigger) |
| 768px - 1024px | Icon strip + labels on hover | Drawer (slide out) |
| >= 1024px | Full width (200px) | Always visible (280px) |

---

## Exit Criteria

- [ ] Shell renders at 375px without horizontal overflow
- [ ] Shell renders at 768px with collapsed sidebars
- [ ] Shell renders at 1024px with visible sidebars
- [ ] Shell renders at 1440px with expanded sidebars
- [ ] Main Canvas fills available space
- [ ] No layout shift when toggling panels

---

## Out of Scope

- Content inside Context Bar (see 0.2)
- Status Bar content (see 0.3)
- ChordOverlay (see 0.4)
- Any view-specific content

---

## Acceptance Tests

1. Load shell at 375px → no horizontal scroll, left/right panels collapsed
2. Resize to 768px → panels remain collapsed but expandable
3. Resize to 1024px → panels visible by default
4. Toggle panel visibility → layout adjusts smoothly
5. Rotate device (mobile) → layout responds correctly
